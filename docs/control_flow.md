# Control Flow

## If Statements

**Else and else-if blocks are currently not supported**

```java
if (x == 0) {
    print("x is 0");
}
```

## While Statements

```java
var i = 10;
while (i != 0) {
    print(i);
    i--;
}
```

## For Statements

```java
for (var i = 0; i != 0; i++) {
    print(i);
}
```
